1.জাভাস্ক্রিপ্ট ব্রাউজারে কিভাবে কাজ করে?

Js কোড যখন ব্রাউজার বা সার্বার পায় তখন কোডটিকে মেশিন ল্যাংগুয়েজে পরিনত করতে হয়। মেশিন ল্যাংগুয়েজে পরিনত করার জন্য বিভিন্ন ব্রাউজারে ভিন্ন ভিন্ন নামের Js engine থাকে (যেমনঃ Google chrome এ থাকে V8 নামক engine, Mozila firefox এ থাকে Spider Monkey নামক engine.) এবং সকল ব্রাউজারের engine গুলো ECMA Script standard অনুসরণ করে। ব্রাউজার engine javascript code কে কম্পাইল করে মেশিন ল্যাংগুয়েজে পরিণত করে এবং আমরা ডিসপ্লেতে তা আউটপুট হিসেবে দেখতে পাই।

2.লোকাল স্টোরেজ এবং সেশন স্টোরেজের মধ্যে পার্থক্য কি?

Local Storage এর মাধ্যমে ডাটা permanent ভাবে স্টোর করে রাখা যায়, প্রয়োজনে ব্যাবহার করা যায় এবং ডিলিট করা যায়। লোকাল স্টোরেজে রাখা ডাটার কোন expiry date নেই। যার করনে যেকোন সময় ডাটা ব্রাউজারের Local storage থেকে ডাটা নিয়ে ব্যাবহার করা যায়। কিন্তু অপরদিকে session Storage এ থাকা ডাটা ব্রাউজার বন্ধ করলে ডাটা গুলো storage থেকে মুছে যায়। তাই session storage এ থাকা ডাটা গুলোকে অস্থায়ী storage system বলা হয়।
